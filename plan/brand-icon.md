# Mnemo Brand Icon

> **Purpose:** Define the Mnemo symbol system and the production constraints for app, tray, sidebar, popup, and empty-state use.
>
> **Primary Mark:** `Trace Loop`

---

## Decision

Mnemo must not use the prior wave-and-dot reference mark. It is retired because it is not original to this project.

The shipping symbol is `Trace Loop`:

- two independent rounded paths that loop toward each other
- one small offset node showing a remembered connection
- thick geometry with open negative space
- no wave silhouette, clipboard, brain, or node mesh

## Why This Direction

The mark needs to communicate:

- memory
- saved reference
- quiet intelligence
- premium utility software

The `Trace Loop` does that better than:

- literal brain outlines
- clipboards
- node-link graphs
- generic AI swirls

## Geometry Rules

- silhouette-first design
- no more than one internal detail
- symmetrical or near-symmetrical balance
- thick enough to survive `16x16`
- designed in monochrome first

The symbol should feel closer to a product emblem than an illustration.

## Usage System

### App Icon

- rounded-square mineral-charcoal container
- sage/mist paths with a muted blush node
- subtle depth allowed, but final shape still reads flat

### Tray Icon

- monochrome symbol only
- no container
- no shading
- optimized for tiny size

### Sidebar Brand

- symbol + `mnemo` wordmark
- left-aligned
- compact horizontal lockup

### Popup Header

- symbol only or symbol + small wordmark
- should not dominate the search field

### Empty-State Watermark

- oversized low-opacity symbol
- never stronger than content

## Rejected Direction

The current network-brain concept is rejected for production because:

- too many interior lines
- poor tiny-size performance
- overly literal “AI graph” association
- tray icon becomes visually muddy
- weak silhouette memory compared to a simpler emblem

## Approved Variant

The selected Trace Loop uses a sage upper path, mist lower path, and blush offset node. Its tray version is monochrome and preserves only the silhouette and node.

## Review Criteria

The final symbol must:

- read instantly at `16x16`
- work in monochrome
- look stable on light and dark backgrounds
- fit naturally inside a rounded-square icon
- avoid resemblance to existing major tech logos

## Asset Targets

Generate:

- three concept boards
- one final app icon
- one final tray icon
- one sidebar lockup mock
- one popup header mock

## Wordmark Guidance

Use a clean geometric sans. The wordmark should not overpower the symbol. Symbol recognition matters more than stylized lettering at this stage.
